Campus clothing swaps with measurable sustainability impact.

Live Demo in 90 sec
If Tinder and Depop had a responsible child, this is it.
The Problem

Closets full. Wallets empty. Planet's dying.

  • Students want new fits without paying new-fit prices.
  • Reselling is slow, expensive, and unsustainable.
  • No one shows you the environmental win from each swap.
Current strategy: "I guess I will just pay for shipping fees."
How STYLR Works

Swipe, Match, Trade, Impact.

  • List clothing in seconds.
  • Swipe through campus listings with a match %.
  • Mutual likes generate item-to-item matches.
  • Confirm trade and track CO2, water, driving saved.
Yes, you can save the Earth in a better jacket.
Demo Time

2-minute walkthrough

Technical Backbone

Fast stack, clean flow

  • Frontend: React + Vite
  • Backend: Spring Boot REST API
  • Data: PostgreSQL cloud + Flyway migrations
  • Matching logic: mutual right-swipes
Why It Wins

Behavior change, but make it fun.

  • Rank system: seedling -> budding plant -> tree.
  • Milestone badges for carbon, water, and miles saved.
  • Campus-focused trust + direct contact options.
Gamified sustainability: dopamine, but biodegradable.
Closing

STYLR makes swapping social, affordable, and measurable.

We are ready for pilot testing on campus communities.
