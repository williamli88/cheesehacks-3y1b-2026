Problem

Campus closet math is bad.

What Students Told Us

  • Too much scrolling.
  • Hard to coordinate safe handoffs.
  • \"I might wear this\" has a 2% success rate.

What We Optimized For

  • Fast yes/no decisions.
  • Campus-only matching.
  • Visible impact after each swap.
Flow

Four screens. No chaos.

Discover

Tech

Simple stack. Clear logic.

Core Logic

  • AI ranking from style/color tags.
  • Double-blind mutual-like matching.
  • Swap impact calculator (water + CO2).

Stack

  • Frontend: React + Vite.
  • Backend: Spring Boot.
  • Data: SQLite now, Postgres-ready.
  • Seeded demo items, no repeats.
